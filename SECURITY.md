# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in MXF, please email Brad Anderson 
directly at BradA1878@pm.me.

Please do NOT create public GitHub issues for security vulnerabilities.

## Response Time

I will acknowledge receipt within 48 hours and provide a detailed response 
within 7 days including:
- Confirmation of the issue
- Estimated timeline for fix
- Credit/acknowledgment preferences

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
